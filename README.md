# Mobile Programming Exercises

## 1.1 Teht1
- Make two buttons at the same horizontal level.

## 1.2 Teht2
- Make different sized boxes on different rows on the screen.  
- Use colors and separators.

## 1.3 Teht3
- Make a simple contact form with `TextFields`.

## 1.4 Teht4
- Make a sport app layout.  
- Use `StringResource` to get a translation for the page.

## 2.1 Teht5
- Make a visitor counter, where you can add and decrease visitors.  
- Add a reset option.

## 2.2 Teht6
- Make a random number generator, where the numbers spawn into three boxes.  
- Numbers cannot be the same between the boxes.

## 2.3 Teht7
- Make a photo carousel with four photos.  
- Add buttons to scroll forward and backward.  
- Add a counter to show which photo is currently being viewed.

## 2.4 Teht8
- Make a game score counter.  
- Add points and fouls for home team and guest team.  
- Make the foul box turn red after three fouls.

## 2.5 Teht9
- Make a dice game.  
- Clicking the button throws the dice.  
- The bigger number is added to the total score.  
- If the dice show the same number, sum those numbers and add to the total.

## 2.6 Teht10
- Make a math quiz game.  
- The app generates random numbers to add together.  
- Correct answers are counted toward the total score.

## 2.7 Teht11
- Make an app where you can list products and their prices.  
- The app stores the inputs to a separate list, which you can access via navigation.
